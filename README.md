# Prepify 🚀

**AI-powered interview preparation platform** that simulates real technical and HR interviews with structured feedback, adaptive questioning, and performance evaluation.

---

## ✨ Overview

Prepify helps you prepare for interviews by providing a **realistic interview environment** powered by AI.
Instead of static questions, it behaves like a real interviewer — asking questions, analyzing responses, and giving feedback.

---

## 🎯 Features

### 🧠 Intelligent Interview System

* DSA, HR, and Mock interview modes
* Dynamic question generation
* Real-time interviewer-like interaction

### ⚙️ Custom Interview Setup

* Difficulty selection (Easy / Medium / Hard)
* Programming language selection (Java, C++, JS, Python)
* Focus areas (Arrays, Strings, Trees, etc.)
* Interview type (Practice / Timed)

### 💬 Interactive Chat Experience

* AI asks one question at a time
* Follow-up questions based on your answers
* Clean and structured responses

### 📊 Performance Evaluation

* End interview with detailed report
* Score out of 10
* Strengths and weaknesses
* Personalized improvement suggestions

### 🔄 Smart Flow

* Guided setup → Interview → Evaluation
* No clutter, no unnecessary prompts
* Real interview-like experience

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/your-username/Prepify.git
cd Prepify/frontend
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run the development server

```bash
npm run dev
```

### 4. Open in browser

```
http://localhost:3000
```
